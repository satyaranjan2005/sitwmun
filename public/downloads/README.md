# Downloads Folder

Place your PDF files here for the Event Timeline buttons.

## Required Files:

1. **gd-prelims-results.pdf** - Group Discussion Prelims Results
2. **country-allotment-results.pdf** - Country Allotment Results  
3. **position-paper-template.pdf** - Position Paper Template

You can change these filenames in the `.env.local` file if needed.

## Note:
This folder is publicly accessible. Files placed here can be downloaded by anyone with the link.
